# Mitata 交互展示

一个基于 Web 的实时眼部文本填充交互展示项目，使用 MediaPipe Face Mesh 进行精确的眼睛和瞳孔检测。

## 功能特点

- 实时摄像头捕捉和人脸检测
- 精确的眼睛和瞳孔区域识别
- 文本填充效果（瞳孔区域自动留白）
- 可调节的放大倍数（1x - 10x）
- 可调节的瞳孔遮罩大小（0x - 2x）
- 自定义填充文本、字体大小和颜色
- 平滑处理减少抖动

## 界面布局

- **上半部分（2/3）**：展示放大的文本填充效果（纯文本，黑色背景）
- **下半部分（1/3）**：实时摄像头捕捉画面

## 快速开始

### 运行项目

由于浏览器安全限制，需要通过本地服务器运行：

```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js
npx http-server -p 8000
```

然后在浏览器中打开 `http://localhost:8000`

### 使用说明

1. 点击"开始捕捉"按钮
2. 允许浏览器访问摄像头
3. 调整控制参数：
   - **填充文本**：输入要填充的文本内容
   - **字体大小**：调整文本大小（4-20）
   - **文本颜色**：设置文本颜色（支持十六进制颜色码）
   - **放大倍数**：调整上半部分的放大倍数（1x-10x）
   - **瞳孔遮罩**：调整瞳孔留白区域大小（0x-2x）

## 技术实现

- **人脸检测**：MediaPipe Face Mesh（468 个面部关键点 + 虹膜检测）
- **图形处理**：HTML5 Canvas
- **平滑算法**：移动平均算法（8 帧缓冲）
- **文本渲染**：按字符级别填充，精确的瞳孔遮罩
