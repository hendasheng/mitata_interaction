# Mitata 交互展示

一个基于 Web 的实时眼部文本填充交互展示项目，使用 MediaPipe Face Mesh 和 MediaPipe Hands 进行精确的眼睛、瞳孔和手势检测。

## 功能特点

- 实时摄像头捕捉和人脸检测
- 精确的眼睛和瞳孔区域识别
- 文本填充效果（瞳孔区域自动留白）
- 独立的放大倍数和清晰度控制
  - **放大倍数**：1x - 20x，控制显示尺寸
  - **清晰度**：0 - 1，控制渲染分辨率（0 映射到 0.2，1 映射到 0.8）
- 眼睛间距调节（-200 到 0，支持眼睛重叠叠加效果）
- 手势控制清晰度
  - **手掌张开上下移动**：实时调节清晰度
  - **握拳**：锁定当前清晰度值
- 可调节的瞳孔遮罩大小（0x - 2x）
- 自定义填充文本、字体大小和颜色
- 平滑处理减少抖动
- F 键全屏切换

## 版本说明

- **当前开发版**：最新功能，包含手势控制

## 快速开始

### 运行项目

由于浏览器安全限制，需要通过本地服务器运行才能访问摄像头。推荐以下方法：

**方法 1：使用 VS Code Live Server 插件（推荐）**
1. 安装 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 插件
2. 在 VS Code 中打开项目文件夹
3. 右键点击 `home.html` 或 `index.html`
4. 选择 "Open with Live Server"
5. 浏览器会自动打开项目

**方法 2：使用 Python 内置服务器**
```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
然后访问 `http://localhost:8000`

**方法 3：使用 Node.js http-server**
```bash
# 安装
npm install -g http-server

# 运行
http-server -p 8000
```
然后访问 `http://localhost:8000`

### 使用说明

1. 通过本地服务器打开 `home.html` 选择版本，或直接打开 `index.html`
2. 点击"开始捕捉"按钮
3. 允许浏览器访问摄像头
4. 调整控制参数：
   - **填充文本**：输入要填充的文本内容
   - **字体大小**：调整文本大小（1-20）
   - **文本颜色**：设置文本颜色（支持十六进制颜色码）
   - **放大倍数**：调整显示尺寸（1x-20x）
   - **清晰度**：调整渲染分辨率（0-1，或用手势控制）
   - **瞳孔遮罩**：调整瞳孔留白区域大小（0x-2x）
   - **眼睛间距**：调整两眼距离（-200 到 0）

### 手势控制

手势控制清晰度的详细步骤：

1. **准备姿势**
   - 将手掌对准摄像头，确保手部在画面中清晰可见
   - 张开手掌，伸展四根手指（食指、中指、无名指、小指）

2. **调节清晰度**
   - 保持手掌张开状态
   - **向上移动手掌**：增加清晰度（文字更清晰）
   - **向下移动手掌**：降低清晰度（文字更模糊）
   - 移动速度适中，系统会实时响应

3. **锁定清晰度**
   - 当调节到满意的清晰度后，**握紧拳头**
   - 系统会锁定当前清晰度值
   - 此时移动手掌不会改变清晰度

4. **解锁继续调节**
   - 再次**张开手掌**即可解锁
   - 可以继续上下移动调节清晰度

**检测原理**：
- 手掌张开：4 根手指中有 3 根以上伸展
- 握拳：4 根手指中只有 1 根或更少伸展
- 使用手腕的垂直位置变化来控制清晰度数值

## 技术实现

- **人脸检测**：MediaPipe Face Mesh（468 个面部关键点 + 虹膜检测）
- **手势检测**：MediaPipe Hands（21 个手部关键点）
- **图形处理**：HTML5 Canvas
- **UI 控件**：Tweakpane
- **平滑算法**：移动平均算法（8 帧缓冲）
- **文本渲染**：按字符级别填充，精确的瞳孔遮罩，支持重叠叠加
