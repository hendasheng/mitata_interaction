# Mitata 交互展示

一个基于 Web 的实时眼部文本填充交互展示项目，使用 MediaPipe Face Mesh 和 MediaPipe Hands 进行精确的眼睛、瞳孔和手势检测。

## 功能特点

- 实时摄像头捕捉和人脸检测
- 精确的眼睛和瞳孔区域识别
- 文本填充效果（瞳孔区域自动留白）
- 独立的放大倍数和清晰度控制
  - **放大倍数**：1x - 20x，控制显示尺寸
  - **清晰度**：0 - 1，控制渲染分辨率（0 映射到 0.2，1 映射到 0.8）
- 眼睛间距调节（-200 到 0，支持眼睛重叠叠加效果）
- 手势控制清晰度
  - **手掌张开上下移动**：实时调节清晰度
  - **握拳**：锁定当前清晰度值
- 可调节的瞳孔遮罩大小（0x - 2x）
- 自定义填充文本、字体大小和颜色
- 平滑处理减少抖动
- F 键全屏切换

## 版本说明

项目包含多个版本，可通过 `home.html` 选择：

- **当前开发版**：最新功能，包含手势控制
- **V2 - 手势控制**：添加手势控制清晰度功能
- **V1 - 放大与清晰度**：独立的放大倍数和清晰度控制

## 界面布局

- **上半部分（2/3）**：展示放大的文本填充效果（纯文本，黑色背景）
- **下半部分（1/3）**：实时摄像头捕捉画面

## 快速开始

### 使用说明

1. 打开 `home.html` 选择版本，或直接打开 `index.html`
2. 点击"开始捕捉"按钮
3. 允许浏览器访问摄像头
4. 调整控制参数：
   - **填充文本**：输入要填充的文本内容
   - **字体大小**：调整文本大小（1-20）
   - **文本颜色**：设置文本颜色（支持十六进制颜色码）
   - **放大倍数**：调整显示尺寸（1x-20x）
   - **清晰度**：调整渲染分辨率（0-1，或用手势控制）
   - **瞳孔遮罩**：调整瞳孔留白区域大小（0x-2x）
   - **眼睛间距**：调整两眼距离（-200 到 0）

### 手势控制

- 将手掌对准摄像头，张开手掌
- 上下移动手掌控制清晰度（向上增加，向下降低）
- 握拳锁定当前清晰度值
- 再次张开手掌可继续调节

## 技术实现

- **人脸检测**：MediaPipe Face Mesh（468 个面部关键点 + 虹膜检测）
- **手势检测**：MediaPipe Hands（21 个手部关键点）
- **图形处理**：HTML5 Canvas
- **UI 控件**：Tweakpane
- **平滑算法**：移动平均算法（8 帧缓冲）
- **文本渲染**：按字符级别填充，精确的瞳孔遮罩，支持重叠叠加
